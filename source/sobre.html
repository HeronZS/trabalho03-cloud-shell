Sobre o Sistema

Este sistema foi desenvolvido como parte do Trabalho 03 da disciplina de Cloud Computing do curso de Sistemas de Informação da Unidavi.

O objetivo é simular um ambiente Linux containerizado para gestão de estoque de uma loja, utilizando Docker, Apache e Shell Script para automação de tarefas operacionais.

Tecnologias Utilizadas

Linux Ubuntu, Docker, Apache, Shell Script, AWS EC2, GitHub e DockerHub.
